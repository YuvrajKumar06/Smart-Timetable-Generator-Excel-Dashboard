# Smart-Timetable-Generator-Excel-Dashboard
📊 Smart Timetable Generator & Workload Management Dashboard
📌 Project Overview

This project demonstrates how Microsoft Excel can be used as a Business Intelligence tool for educational institutions.

The dashboard helps analyze timetable allocation, classroom utilization, teacher workload, subject distribution, and scheduling efficiency through dynamic dashboards, Pivot Tables, slicers, and KPI cards.

The project was built entirely in Microsoft Excel without using Power Query or VBA, showcasing advanced spreadsheet analytics techniques.

🎯 Objectives
-Create a centralized Master Data table
-Analyze faculty workload
-Monitor classroom utilization
-Track subject distribution
-Visualize timetable trends
-Build interactive dashboards using Pivot Charts and Slicers
-Provide scheduling insights for decision making
🛠 Tools Used
-Microsoft Excel
-Pivot Tables
-Pivot Charts
-Slicers
-Conditional Formatting
-Excel Formulas
-Data Validation
